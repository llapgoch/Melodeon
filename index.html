John Lewis style multi level nested navigation

A menu inspired by the John Lewis mobile website. Uses a basic nested unordered list and allows you to drill down to sections. It hides other sibling options making it easier to interpret the menu. Add .js-menu to the ul to get started. Add .js-menu-default to specify what to open on launch. It must be an unordered list for now. It will infinitely nest as you please.
